Unwrapping Formula 1

Unwrapping the Formula 1 Prediction Model

Model Overview

This Formula 1 race winner prediction model uses advanced machine learning techniques to forecast the outcome of F1 races. It's built on historical data from 2014 to 2023 and employs a Gradient Boosting Classifier (XGBoost) for high accuracy predictions.

Data Sources

  • race_results.csv: Race outcomes and details
  • qualifying_results.csv: Qualifying performance data
  • driver_standings.csv: Cumulative driver performance
  • constructor_standings.csv: Team performance metrics

Data Preprocessing

Key Steps:

  • Merging multiple data sources
  • Handling missing values with forward fill and imputation
  • Feature engineering (e.g., rolling averages, position differences)
  • Encoding categorical variables
  • Normalizing numerical features with StandardScaler

Feature Selection

We used Random Forest feature importance to identify the most predictive features, including:

  • Recent form
  • Qualifying position
  • Historical performance at the specific track
  • Current season performance

Model Details

Algorithm: XGBoost (Gradient Boosting Classifier)

Hyperparameters:
  • max_depth: 6
  • n_estimators: 200
  • learning_rate: 0.1
  • subsample: 0.8
  • colsample_bytree: 0.8

Training Process

  1. Data split: 80% training, 20% testing
  2. Stratified sampling for balanced race outcomes
  3. 5-fold cross-validation
  4. Hyperparameter tuning via grid search

Model Performance

Accuracy on test set: 92%

Key metrics: F1-score, ROC-AUC

The model demonstrates strong predictive power, considering the inherent unpredictability of F1 races.

Limitations and Future Improvements

  • Incorporating real-time data for more dynamic predictions
  • Expanding the feature set to include weather conditions and car performance metrics
  • Implementing ensemble methods to further improve accuracy
  • Developing a system for continuous model retraining with new race data
